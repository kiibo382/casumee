# Casumee
## About
casualに人とチャットまたはwebミーティングができるアプリケーション

## Requirement
 - node v14.4.0
 - mongodb v4.4.2
 - redis v6.2.1
 - npm install -g nodemon

## Backend
### Build
 - make dev-init
 - make build-local
### Run
 - make run-local-db
 - make run-local-server
