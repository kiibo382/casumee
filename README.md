# Casumee
## About
casualに人とチャットまたはwebミーティングができるアプリケーション

## Requirement
 - node v14.4.0

## Backend
### Build
 - make dev-init
 - make build-local
### Run
 - make run-local-db
 - make run-local-dev-server
